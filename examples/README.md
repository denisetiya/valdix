# Examples

Run with: `node examples/01-basic.mjs` (after `npm run build`).

| File | What it shows |
|------|---------------|
| `01-basic.mjs` | User signup form with custom error messages and superRefine |
| `02-recursive.mjs` | Recursive type (tree) with `v.lazy()` |
| `03-json-schema.mjs` | Export JSON Schema for OpenAPI/JSON Schema validators |
| `04-i18n.mjs` | Switch between en/id/jp at runtime |
